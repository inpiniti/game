# 맵 편집법 (Tiled)

이 게임의 지형(물·잔디·언덕·물거품)은 **Tiled** 라는 무료 맵 에디터로 그립니다.
Tiled에서 **눈으로 보며 칠하고 저장 → 브라우저 새로고침** 하면 게임에 바로 반영됩니다.

---

## 1. Tiled 설치

1. https://www.mapeditor.org 접속 → **Download** 클릭
2. Windows용 설치 (무료. "결제 금액"을 **$0** 으로 두고 받을 수 있습니다)
3. 설치 후 실행

## 2. 맵 열기

Tiled에서 **File → Open** →
`rilybox-game/map/level1.json` 선택

열면:
- 가운데: 맵 (지금은 물 위에 잔디 섬 하나)
- 오른쪽 위 **Layers(레이어)** 패널: `elevation / ground / foam / water` 4개
- 오른쪽 아래 **Tilesets(타일셋)** 패널: `grass / water / elevation / foam` 4개

## 3. 칠하는 법 (기본)

1. **Layers 패널**에서 그릴 레이어를 클릭해 선택
   - 물 = `water`, 잔디 = `ground`, 언덕 = `elevation`, 물거품 = `foam`
2. **Tilesets 패널**에서 칠할 타일 한 칸을 클릭
3. 상단 툴바에서 **Stamp Brush(도장, 단축키 B)** 선택 후, 맵에서 드래그하며 칠하기
4. 지우개는 **Eraser(E)**, 사각형 채우기는 **Bucket Fill(F)**

> 💡 팁: **View → Show Grid** 로 격자를 켜면 칸이 보여 편합니다.
> 레이어를 헷갈리지 않게 **Highlight Current Layer** 도 켜두면 좋아요.

## 4. 레이어별 타일 가이드

- **grass(잔디)** — 3×3 묶음이 섬 테두리 세트입니다.
  가운데 칸 = 안쪽 채움, 바깥 8칸 = 물과 맞물리는 가장자리(모서리/코너).
  섬을 넓히려면 안쪽은 가운데 타일, 가장자리는 테두리 타일로 칠하세요.
- **water(물)** — 파란 물 한 칸. 잔디를 지운 자리에 깔거나 배경용.
- **elevation(언덕/절벽)** — 높이(언덕)를 만드는 타일. `elevation` 레이어에 칠합니다.
- **foam(물거품)** — 해안선을 따라 두르는 물거품. `foam` 레이어에 칠합니다.

## 4-1. 타일셋 (이미 전부 추가돼 있음)

**두 Terrain 폴더의 모든 타일(70개)이 이미 맵에 embed돼 있습니다.**
Tiled의 **Tilesets** 패널에 grass/water/elevation/foam + Free Pack의 Tilemap_color1~5,
Water Foam, 그리고 덤불·구름·바위·나무·양·금 등 장식/자원까지 전부 나타납니다.
→ **수동으로 추가할 필요 없이** 바로 골라서 칠하면 됩니다.

> 타일셋 목록은 `js/tileset-manifest.js` 에 자동 생성돼 있고, 게임이 이걸 보고
> 모든 이미지를 자동 로드합니다. (맵을 다시 생성하면 이 파일도 같이 갱신됩니다.)

### 그 외 폴더(Factions 등)의 타일을 쓰고 싶다면

Terrain 밖의 이미지를 쓰려면 Tiled에서 직접 추가하세요:

1. 오른쪽 아래 **Tilesets** 패널 → **New Tileset**(새 타일셋) 버튼
2. 설정:
   - **Name(이름)**: 파일명 그대로 두기 (예: `Tilemap_color3`) ← ⚠️ 바꾸지 마세요
   - **Type**: *Based on Tileset Image*
   - **Source**: Browse → `assets/Tiny Swords (Free Pack)/Tiny Swords (Free Pack)/Terrain/Tileset/Tilemap_color3.png`
   - **Tile width / height**: `64` / `64`, Margin `0`, Spacing `0`
   - ✅ **Embed in map** 체크 (안 하면 게임에서 안 보임!)
3. OK → 이제 타일셋 패널에 나타남 → 원하는 레이어에 칠하기

> 제가 게임 쪽에 미리 배선해둔 타일셋(이름 그대로 추가하면 자동 렌더):
> `Tilemap_color1`, `Tilemap_color3`, `Water Background color`, `Water Foam`.
> 다른 파일을 쓰고 싶으면 알려주세요 — 게임 코드에 연결해 드립니다.

## 5. 저장 & 확인

1. **Ctrl + S** 로 저장 (그대로 `level1.json` 에 덮어써집니다)
2. 브라우저에서 **Ctrl + Shift + R** (새로고침)
3. 바뀐 지형이 게임에 나타납니다 🎉

## ⚠️ 지금 단계의 한계 (알아두기)

- **캐릭터 이동 범위**는 아직 시작 섬 사각형(타일 2~17 × 2~10)에 고정돼 있습니다.
  섬 모양을 크게 바꾸면 캐릭터가 물 위를 걷거나, 잔디인데 막힐 수 있어요.
  → 섬 모양을 확정하면 알려주세요. **타일 기반 충돌**(물·절벽은 못 지나감)로 바꿔드립니다.
- **나무·금·양·건물** 위치도 아직 코드에 박혀 있습니다.
  섬을 옮기면 이것들이 물 위에 뜰 수 있어요. → 나중에 **Tiled 오브젝트 레이어**로 옮겨서
  맵에서 직접 배치할 수 있게 바꿀 수 있습니다.
