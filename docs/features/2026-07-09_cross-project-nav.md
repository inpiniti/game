# 기능: 3개 프로젝트 로고 네비게이션 — notepad·trends·game 왕복 링크

> 작성일: 2026-07-09 · 분류: **소형** — 로고 href 링크 추가만 (3개 파일, 1단계)

## 왜

notepad와 trends 사이에는 이미 로고 부분에 왕복 링크가 있어서 오갈 수 있었다. game 프로젝트도 3개 중심 프로젝트인데 네비게이션에서 빠져있어서, 3개 프로젝트 모두 로고에서 서로 이동 가능하도록 한다.

## 작업

| 단계 | 작업 | 사용 스킬 | 에이전트 모델 (effort) | 완료 기준 |
|---|---|---|---|---|
| 1 | notepad·trends·game 각 프로젝트의 로고(Header) 컴포넌트에 3개 프로젝트 링크 추가 — 로고 클릭 시 프로젝트 선택 메뉴 또는 직접 링크. notepad↔trends 기존 패턴 확인 후 game도 동일 방식 적용 | — | Sonnet (medium, thinking O) | 3개 프로젝트 로고에서 다른 2개 프로젝트로 이동 가능 확인 |

## 완료 기준

- notepad 로고 클릭 → trends·game 링크 표시/제공
- trends 로고 클릭 → notepad·game 링크 표시/제공  
- game 로고 클릭 → notepad·trends 링크 표시/제공
- 각 링크 클릭 시 해당 프로젝트로 이동
