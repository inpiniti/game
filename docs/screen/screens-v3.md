# 화면 구성안 v3 — 딸기 서바이벌

> 게임명: **딸기 서바이벌** (영문: `ddalki-survival`, 🍓)
>
> 작성일: 2026-07-08 · [v2](./screens-v2.md) 대체 · 설계 기준: [docs/design/supabase-quiz-fsd-v8.md](../design/supabase-quiz-fsd-v8.md)
> 대상: 웹(데스크톱) · 태블릿 · 모바일 — 하나의 반응형 웹앱
>
> **v2 대비 변경점 (그릴링 결과 반영)**
> 1. **게임은 서바이벌 하나** — 단어장 게임(play-vocab)·복습(review) 화면 삭제, 방식 선택 모달 삭제(방향만 선택)
> 2. **일시정지 메뉴 추가** — 그만하기 = 죽은 것으로 간주(기록 저장)
> 3. **퀴즈 오버레이에 직접입력 변형 추가** — 익은 단어(2회 이상 정답)는 주관식, 복수 뜻이면 입력창 N개
> 4. **이메일 확인 안내 화면 추가** (Supabase 이메일 인증 ON 유지)
> 5. **문제집 상세([문제집 보기]) 화면 추가** — 단어 목록 + 학습 상태 + 개인 문제집 수정 진입
> 6. **프로필 바텀시트** (닉네임 변경·로그아웃) — 헤더 닉네임 탭으로 진입
> 7. 랭킹은 **내 국가 기본** + 국기 이모지 대신 국가 코드 텍스트 (Windows 국기 이모지 미지원)

## 1. 공통 규칙

### 반응형 브레이크포인트

| 구간 | 폭 | 레이아웃 |
|---|---|---|
| 모바일 | ~767px | 1열, 하단 탭바 |
| 태블릿 | 768~1023px | 2열 그리드, 상단 헤더 |
| 데스크톱 | 1024px~ | 3열 그리드, 상단 헤더 |

### 네비게이션

- **데스크톱·태블릿**: 상단 헤더 (로고 / 문제집 / 랭킹 / 기록 / [관리: admin만] / 닉네임)
- **모바일**: 상단 간소 헤더(로고 + 닉네임) + **하단 탭바** `[홈] [문제집] [랭킹] [기록]`
  - 게임 중에는 탭바·헤더 숨김 (전체 화면)
- **닉네임 탭 → 프로필 바텀시트** (닉네임 변경·로그아웃) — 모바일/데스크톱 공통
- 로그인/회원가입/이메일 확인 화면에는 헤더·탭바 없음

### 터치·게임 대응

- 터치 타깃 최소 44px, 퀴즈 선택지 버튼 56px 이상
- 서바이벌: 데스크톱 키보드(방향키) / **터치 기기는 가상 조이스틱**
- Phaser `Scale.RESIZE` — 세로·가로 모두 플레이 가능, iOS 세이프에어리어 패딩
- **퀴즈 오버레이·일시정지 메뉴는 DOM(React) 레이어** — 캔버스 위에 겹침 (직접입력 키보드 대응)
- 테마: 다크 단일 테마

### 화면 흐름 (v3)

```
로그인 ⇄ 회원가입 → 이메일 확인 안내 → (메일 링크 클릭) → 로그인
   ↓ 성공
 로비 ──→ 문제집 선택 ──→ (문제집 상세) ──→ 방향 선택 시트 ──→ 서바이벌 플레이
   │                                                          │ 퀴즈 오버레이 반복
   ├──→ 랭킹 · 기록                                            │ 죽음/그만하기
   └──→ 관리 (admin만)                                         ↓
                                        결과 화면 ──→ [다시하기] / [로비로]
```

### 라우트 매핑

| 화면 | 라우트 | 가드 |
|---|---|---|
| 로그인 / 회원가입 / 이메일 확인 | `/login` `/signup` `/verify-email` | 비로그인만 |
| 로비 | `/` | RequireAuth |
| 문제집 선택 / 상세 | `/sets` `/sets/:setId` | RequireAuth |
| 서바이벌 | `/play/:setId` | RequireAuth |
| 결과 | `/result` (직전 세션 state — **새로고침·직접 진입 시 로비로 리다이렉트**) | RequireAuth |
| 기록 / 랭킹 | `/history` `/ranking` | RequireAuth |
| 관리 | `/admin/sets` `/admin/upload` | RequireAdmin |

## 6. 문제집 선택 `/sets`

```
+------------------------------------------------------------------+
| 언어:  [영어단어] [일본어단어] [일반]      ← learn_lang 탭 (있는 것만) |
| 구분:  [공식] [내 문제집]                        [+ 업로드]        |
+------------------------------------------------------------------+
|  +------------------+  +------------------+  +------------------+ |
|  | 기초 영단어        |  | 중1 영단어        |  | 토익 필수 500     | |
|  | 단어 20개         |  | 단어 120개        |  | 단어 500개        | |
|  | 학습 15/20        |  | 학습 0/120       |  | 학습 48/500      | |
|  |                  |  |                  |  |                  | |
|  | [문제집 보기] [시작] |  | [문제집 보기] [시작] |  | [문제집 보기] [시작] | |
|  +------------------+  +------------------+  +------------------+ |
+------------------------------------------------------------------+
```

- "학습 n/N" = SRS 진행(노출된 카드 수 / 전체).
- [시작] → 방향 선택 시트(8절) → 플레이. [문제집 보기] → 상세(7절).
- [내 문제집] 탭: 카드에 [삭제] 추가 (수정은 상세에서).
- 모바일: 1열 리스트, 동일 구성.

## 7. 문제집 상세 `/sets/:setId` (신규 — "문제집 보기")

```
+------------------------------------------------------------------+
| [← 뒤로]   기초 영단어                            (내 것이면) [수정] |
|            영어단어 · 단어 20개 · 학습 15/20                       |
+------------------------------------------------------------------+
| ⚠ 자주 틀리는 단어                                                |
|  anywhere   어딘가에;아무데도;어디든지        오답 12회  💬         |
|  run        달리다;뛰다                     오답 9회   💬         |
+------------------------------------------------------------------+
| 전체 단어                                                         |
|  단어        뜻                     상태                          |
|  apple      사과                   3일 뒤 복습                    |
|  book       책                     학습 중                        |
|  water      물                     미학습                         |
|  happy      행복한;기쁜             30일 뒤 복습                   |
|  ...                                                             |
+------------------------------------------------------------------+
|  [               게임 시작                ]   ← 하단 고정 CTA      |
+------------------------------------------------------------------+
```

- 행 탭 → 예문(💬) 펼침.
- 상태 = `srs_progress` 기반: 미학습 / 학습 중 / n일 뒤 복습 / 오답 n회.
- "자주 틀리는 단어" = lapses 높은 순 상위 몇 개 (취약 단어 확인 겸용).
- 개인 문제집 [수정]: 단어 행 추가·편집·삭제 (같은 화면에서 편집 모드 전환).

## 9. 서바이벌 플레이 `/play/:setId`

### 플레이 중 (Phaser 캔버스 전체 화면)

```
+------------------------------------+
| ♥♥♥  점수 1,250  퀴즈 4/5     [⏸] |
+------------------------------------+
|                                    |
|         🧟      🐺                 |
|              🛡←기사               |
|         🧟          🧟             |
|                                    |
|   (+)                              |
|  조이스틱 ※터치 기기만              |
+------------------------------------+
```

### 퀴즈 오버레이 — 3지선다 (낯선 단어: repetition 0~1)

DOM 레이어. 게임 일시정지 상태.

```
+------------------------------------+
|      (반투명 어두운 배경 72%)        |
|                                    |
|        anywhere 의 뜻은?            |
|                                    |
|  [          어딘가에           ]   |
|  [          아무데도           ]   |
|  [          오답선지           ]   |
|                                    |
+------------------------------------+
```

정답 공개 (0.85초 후 게임 복귀):

```
|  [✅         어딘가에          ]   |  ← 정답 변형 전부 초록 강조
|  [✅         아무데도          ]   |
|  [❌         오답선지          ]   |
|                                    |
|  💬 You can sit anywhere.          |  ← 예문 (있으면)
|  🎁 축복 획득: 공격력 +10%          |  ← 정답 보상 / 오답이면 보상 없음
```

### 퀴즈 오버레이 — 직접입력 (익은 단어: repetition ≥ 2, 자동 승급)

복수 뜻이면 입력창 N개. 순서 무관, 전부 맞아야 정답(보상).

```
+------------------------------------+
|        anywhere 의 뜻은?            |
|        ※ 뜻 3개를 입력              |
|                                    |
|   답 1  ______________________     |
|   답 2  ______________________     |
|   답 3  ______________________     |
|                                    |
|   [           제출            ]    |
+------------------------------------+
```

채점 피드백 (칸별 ✓/✗ + 못 맞힌 변형 공개):

```
|   답 1  어딘가에               ✓   |
|   답 2  어디에나               ✗   |
|   답 3  아무데도               ✓   |
|                                    |
|   못 맞힌 뜻:  어디든지             |
|   💬 You can sit anywhere.         |
```

- Enter = 다음 칸, 마지막 칸 Enter = 제출. 모바일은 키보드 위에 입력창이 보이도록 스크롤.

### 일시정지 메뉴 (신규 — [⏸] 탭)

```
+------------------------------------+
|            일시정지                 |
+------------------------------------+
|                                    |
|  [         계속하기            ]   |
|                                    |
|  [         그만하기            ]   |
|   지금 점수(1,250점)로 기록돼요     |
|                                    |
+------------------------------------+
```

- **그만하기 = 죽은 것으로 간주** → 게임오버 처리(기록 저장) → 결과 화면.
- 딤 탭/뒤로 가기 = 계속하기.

## 11. 기록 `/history` · 랭킹 `/ranking`

### 기록 (v2와 동일 — 게임 종류 열 삭제)

```
데스크톱: 날짜 | 문제집 | 점수 | 처치 | 정답률   (행 탭 → 틀린 단어 펼침)
모바일:   카드 리스트 (문제집 · 점수 · 날짜 · 정답률)
```

### 랭킹 — 내 국가 기본

```
+------------------------------------------------------------------+
| 탭: [문제집별] [전체] [국가 대항]        범위: [내 국가(KR) | 전체] |
+------------------------------------------------------------------+
| (문제집별 탭)  문제집: [기초 영단어 ▼]                             |
|                                                                  |
|  순위   닉네임            국가    점수                             |
|   1     wordmaster       KR     1,850                            |
|   2     Taro             JP     1,720                            |
|   3     철수              KR     1,500                            |
+------------------------------------------------------------------+
|  ▶ 내 순위: 15위 · 830점                          ← 하단 고정 강조 |
+------------------------------------------------------------------+
```

- **기본 범위 = 내 국가** (profiles.country). [전체]로 토글 가능.
- 국가는 **코드 텍스트(KR, JP…)** — Windows 브라우저가 국기 이모지를 렌더링하지 못함.
- 전체 탭: 평균점 + 참여 문제집 수. 국가 대항 탭: 국가 / 평균점 / 참여 인원.

## 12. 프로필 바텀시트 (신규 — 헤더 닉네임 탭)

```
+------------------------------------+
|  프로필                        [X] |
+------------------------------------+
|  닉네임   [ 철수          ] [변경]  |
|  국가     KR (대한민국)             |
|  이메일   user@example.com         |
|                                    |
|  [          로그아웃           ]   |
+------------------------------------+
```

- 닉네임 인라인 변경(profiles.nickname). 국가 변경도 허용(검색 select).
- 로그아웃 → 로그인 화면.

## 13. 관리 `/admin/sets` `/admin/upload` (v2와 동일)

- 목록: 국가·언어 필터 + [수정][삭제], [개인 문제집 승격] 탭(승격 시 국가·언어 지정 모달)
- 업로드: 제목 + 국가 + 언어(또는 일반) + CSV/JSON(예문 3열) + 파싱 미리보기·행 단위 오류 표시
- 개인 업로드도 같은 폼 재사용 (국가 행 없음)
